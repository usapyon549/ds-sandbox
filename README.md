# ds-sandbox

機械学習・データサイエンスを中心とした技術検証用リポジトリです。

portfolio や ml-lab のような完成度を重視したプロジェクトとは異なり、興味を持ったテーマや技術を気軽に試しながら学習記録として残すことを目的としています。

Notebook ベースの小規模な分析や実験が中心で、データ分析、機械学習、LLMなど幅広いテーマを扱います。
